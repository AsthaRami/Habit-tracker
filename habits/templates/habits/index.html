💡 Why This Project?

Key highlights of this Django project

🏗️
MTV Architecture

Follows Django's Model-Template-View pattern with clean separation of concerns.

🗄️
SQLite Database

Uses SQLite with proper ForeignKey relationships between User, Habit and HabitLog models.

🔒
Secure & Safe

CSRF protection, password hashing, login_required decorators for complete security.

📊
Data Visualization

Chart.js integration for beautiful donut and bar charts showing real-time data.

👥
Multi-User Support

Each user has their own private data — complete data isolation and privacy.

⚡
Fast & Efficient

Optimized database queries using Django ORM for fast and efficient data retrieval.
